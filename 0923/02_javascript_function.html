02_javascript_function(함수)

함수란?(function)

함수는 일련의 내용을 삽입하여 사용자가 언제든 호출하여 내용을 실행할 수 있는 실행문이다.

코드의 중복제거 및 재사용이 용이하다

유지보수의 용이성이 높다.

함수의 종류는 내장함수 - prompt('사용자가 입력'), alert('경고메시지').....와 사용자 정의 함수가 있다.

        1.매개변수가 없는 일반함수
         function 함수명(){
            실행할 내용;
         }

         2.매개변수가 있는 함수
         function 함수명(변수1, 변수2, 변수3.....){
            실행할 내용;
         }

         3.리턴값이 있는 함수
         function 함수명(변수1, 변수2, 변수3.....){
            실행할 내용;
            return 실행결과;
         }

문1, 사용자가 입력한 숫자를 변수에 대입하여 1부터 n까지의 합계를 구하기
